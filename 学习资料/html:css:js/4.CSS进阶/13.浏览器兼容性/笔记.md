IE中，CSS的特殊前缀

- *属性，兼容IE5、IE6、IE7
- _属性，兼容IE5～IE6

## 渐进增强 和 优雅降级

两种解决兼容性的思路，会改变代码的风格

- 渐进增强：先适应大部分浏览器，然后针对新版本浏览器加入新的样式

书写代码时，先避免书写有兼容性问题的代码，完成后，再逐步加入新标准中的代码

- 优雅降级，先制作完整的代码，然后针对低版本浏览器进行特殊处理

书写代码时，先不用特别在意兼容性，完成整个功能后，再针对低版本浏览器处理样式
